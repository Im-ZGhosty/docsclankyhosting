# Cambiar versión de Java

¿Que versión de Java usar para mi servidor de Minecraft? Aquí te dejamos una lista de las versiones compatibles con cada versión de Minecraft.

## Lista de versiones compatibles.

- **Minecraft 1.18** y superior necesita **java 17**

- **Minecraft 1.17** y superior necesita **java 16**

- **Minecraft 1.16.5** e inferior usar **java 8 o 11**

- **Minecraft 1.12** e inferior usar **java 8**

## Instrucciones para cambiar versión de Java.

- 1. Dirigete a la consola de tu servidor y da clic en la sección StartUp.

  ![](https://105539378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F09f6qVEsCJT9tRm61hbu%2Fuploads%2FwkevEl3loxtyQkTz35gf%2Fimage.png?alt=media&token=d6561f76-7a79-4a08-bffc-a9f3888b3a15)

  ​

- 2. Selecciona la versión de Java a usar en la opción **DOCKER IMAGE.**

  ![](https://105539378-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F09f6qVEsCJT9tRm61hbu%2Fuploads%2FM44T0Dug2Ri1MgbNu0N6%2FScreenshot%202022-05-11%20000045.png?alt=media&token=e6b9889b-514c-4afa-a888-b8889e1fe0c7)

- 3. Regresa a tu consola y enciende o reinicia tu servidor.
