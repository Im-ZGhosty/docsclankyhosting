# Administrador de archivos

Dentro del panel contaras con un administrador de archivos propio en el cual podrás modificar, subir y eliminar archivos.

::: tip ¿Tienes problemas para subir una carpeta o la carga es demasiado lenta?
Te aconsejamos comprimir tus archivos/carpetas en un solo archivo y en formato .**zip** o **.tar** despues de eso sube nuevamente tus archivos y dentro del Game Panel usa la opción "Unarchive" para deescomprimir los archivos.
:::

## Acceder a tus archivos mediante SFTP

Para acceder a tus archivos mediante SFTP (Secure File Transfer Protocol) deberás ir a la sección **Settings** en tu **Game Panel**. Dentro de esta sección podrás ver un recuadro con los datos necesarios para conectarte.
